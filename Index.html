BROWSER NOT SUPPORTED

QUESTLOAD requires advanced USB access (WebUSB) and a secure context.

  • WebUSB API support (Chrome / Edge / Chromium-based)
  • HTTPS or localhost (secure context)
  • Developer mode enabled on your Quest headset
    • ✓ Google Chrome ✓ Microsoft Edge ✗ Firefox ✗ Safari

    Firefox and Safari do not implement WebUSB.
    Please switch to Chrome, Edge, Brave, or Opera to sideload APK/OBB files directly.
